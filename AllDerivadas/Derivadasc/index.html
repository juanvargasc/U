Idea central

Introducción a las derivadas

Una derivada mide qué tan rápido cambia una cantidad respecto a otra. En cálculo, nos dice la tasa de cambio instantánea de una función.

Definición intuitiva

Si una función describe una posición, su derivada describe velocidad. Si describe costo, su derivada aproxima el costo marginal.

f'(x) ≈ cambio en y / cambio en x

Cuando el intervalo se hace cada vez más pequeño, la razón promedio se convierte en razón instantánea.

Definición formal

La derivada de f en x se define como el límite del cociente incremental:

f'(x) = limh→0 [f(x + h) - f(x)] / h

Si ese límite existe, la función es derivable en ese punto.

Interpretación geométrica

La derivada en un punto es la pendiente de la recta tangente a la curva. En la gráfica, mueve el control para ver cómo cambia la tangente de f(x)=x².

Herramientas

Reglas de derivación

Las reglas permiten derivar funciones compuestas por sumas, productos, cocientes y funciones dentro de funciones.

Regla del producto

(uv)' = u'v + uv'

Ejemplo: y = x² sin(x)

  1. u=x², v=sin(x)
  2. u'=2x, v'=cos(x)
  3. y'=2x sin(x)+x² cos(x)

Regla del cociente

(u/v)' = (u'v - uv') / v²

Ejemplo: y = (x²+1)/x

  1. u=x²+1, v=x
  2. u'=2x, v'=1
  3. y'=[2x·x-(x²+1)]/x²

Regla de la cadena

[f(g(x))]' = f'(g(x))g'(x)

Ejemplo: y = sin(x²)

  1. Función externa: sin(u)
  2. Función interna: u=x²
  3. y'=cos(x²)·2x

Ejemplo paso a paso: y = x cos(x)

  1. Identifica producto: u=x y v=cos(x).
  2. Deriva cada parte: u'=1 y v'=-sin(x).
  3. Aplica (uv)' = u'v + uv'.
  4. Resultado: y' = cos(x) - x sin(x).
Ángulos desde razones

Derivadas trigonométricas inversas

Las funciones inversas trigonométricas responden: ¿qué ángulo produce esta razón? Sus derivadas suelen incluir raíces y restricciones de dominio.

arcsin(x)

d/dx[arcsin(x)] = 1 / √(1 - x²)

Ejemplo: d/dx[arcsin(2x)] = 2 / √(1 - 4x²).

arccos(x)

d/dx[arccos(x)] = -1 / √(1 - x²)

Ejemplo: d/dx[arccos(x²)] = -2x / √(1 - x⁴).

arctan(x)

d/dx[arctan(x)] = 1 / (1 + x²)

Ejemplo: d/dx[arctan(3x)] = 3 / (1 + 9x²).

Cuando y está mezclada con x

Diferenciación implícita

Se usa cuando una ecuación define una relación entre x e y sin despejar y de forma directa.

Ejemplo guiado: x² + y² = 25

  1. Deriva ambos lados respecto a x.
  2. d/dx[x²] = 2x.
  3. d/dx[y²] = 2y · y' por regla de la cadena.
  4. La ecuación queda 2x + 2y y' = 0.
  5. Despeja: y' = -x/y.

Para x³ + y³ = 6xy:

3x² + 3y²y' = 6y + 6xy'

y'(3y² - 6x) = 6y - 3x²

y' = (6y - 3x²) / (3y² - 6x)

Potencias complicadas

Diferenciación logarítmica

Conviene cuando hay productos largos, cocientes con muchas partes o variables en base y exponente, como y = xˣ.

Cuándo se usa

Úsala si la función tiene factores multiplicados, raíces, exponentes variables o una forma que se simplifica con propiedades de logaritmos.

Procedimiento

  1. Escribe y=f(x).
  2. Toma ln en ambos lados.
  3. Expande con propiedades de logaritmos.
  4. Deriva implícitamente.
  5. Despeja y' y sustituye y.

Ejemplo: y = xˣ

  1. ln(y) = ln(xˣ) = x ln(x).
  2. Deriva: y'/y = ln(x) + 1.
  3. Multiplica por y: y' = y[ln(x)+1].
  4. Sustituye y: y' = xˣ[ln(x)+1].
